# 开发命令速查

## 后端（项目根目录）

```bash
# 编译（静默，只显示错误）
mvn compile -q

# 启动应用
mvn spring-boot:run

# 打包
mvn package -q

# 运行测试
mvn test
```

## 前端（`frontend/` 目录）

```bash
# 安装依赖
npm install

# 开发模式（端口 3000，自动代理 API 到 localhost:8080）
npm run dev

# 构建（产物输出到 ../src/main/resources/static/）
npm run build

# 类型检查
npx vue-tsc --noEmit
```
